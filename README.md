# 🎓 Projeto em Linguagem C --- Faculdade Estácio

Curso: Análise e Desenvolvimento de Sistemas

Este repositório contém projetos desenvolvidos em **linguagem C** como
parte das atividades acadêmicas da disciplina de Programação.

## 🛠 Tecnologias Utilizadas

-   **Linguagem C**\
-   **Compilador GCC** (ou similar)\
-   **IDE/Editor**: Code::Blocks, Dev-C++, VS Code ou equivalente

## 📋 Funcionalidades dos Projetos

### 🚢 Jogo Batalha Naval

-   Modos de dificuldade: **Novato**, **Aventureiro** e **Mestre**\
-   Percurso e posicionamento das embarcações no tabuleiro\
-   Sistema de acertos e erros

### 🃏 Jogo Cartas Super Trunfo

-   Modos de dificuldade: **Novato**, **Aventureiro** e **Mestre**\
-   **Cadastro de cartas** personalizadas\
-   Escolha de propriedades para disputa\
-   Exibição do **vencedor** da rodada

### ♟️ Jogo de Xadrez

-   Modos de dificuldade: **Novato**, **Aventureiro** e **Mestre**\
-   Percurso das peças no tabuleiro\
-   Regras básicas implementadas para movimentação

## 🚀 Como Executar os Projetos

1.  Compile os arquivos `.c` utilizando um compilador C (ex.: GCC):

    ``` bash
    gcc nome_do_arquivo.c -o nome_do_programa
    ```

2.  Execute o programa compilado:

    ``` bash
    ./nome_do_programa
    ```

3.  Siga as instruções exibidas no terminal para jogar.

## 📚 Objetivo Acadêmico

Esses projetos têm como finalidade exercitar a lógica de programação e a
aplicação de conceitos fundamentais da linguagem C, explorando
**estruturas de repetição, condicionais, arrays, funções e estruturas de
dados**.
